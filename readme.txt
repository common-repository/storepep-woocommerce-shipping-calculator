== Description ==

= Live WooCommerce Shipping Rates =

StorePep's WooCommerce Shipping calculator allows you to seamlessly integrate all the top shipping carriers like FedEx, UPS, Stamps.com(USPS) and DHL with your woocommerce store. This plugin has the ability to integrate any or all of these shipping carriers and display live shipping rates on the cart and checkout pages. You also have the flexibility to define flat rates and free shipping as per your business needs.

The live rates are accurately displayed on the cart and checkout pages based on customer’s location, shipping class, total weight and total quantity of the shipment.

<strong>This plugin comes with a FREE StorePep.com account with 10,000 API calls per month. If you need Unlimited API access, Printing Shipping Labels in bulk or Live Shipment tracking notifications for customers, subscribe to [StorePep](https://www.storepep.com/#pricing) with a minimal $9/month.</strong> 

= Printing Shipping Labels in bulk (Premium feature) =

Import orders from your woocommerce store and generate 100s of labels in a click. Not just that, by allowing to generate manifests and book pick up, StorePep takes care of your delivery process end to end. 

= Shipment tracking notifications for customers (Premium feature) =

Provide live shipment tracking to your customers.Your customers receive an update everytime there is a progress or change in the shipment status. With this feature, StorePep is a one stop solution for your shipment process.


= Export Tracking Details back to your woocommerce store  (Premium feature)  =
The plugin saves the tracking details back to your WooCommerce orders and changes the order status to completed as soon as a manifest is generated. This way Storepep covers all the aspects of automating the shipping process.


= HOW TO USE THE PLUGIN = 
* Install the plugin and go to WooCommerce –> Settings –> Shipping –> StorePep Shipping
* Provide your email ID and click "Signup StorePep.com & Get API Keys".This will create a FREE StorePep account and the integration is completed.
* Enable “Realtime Rates”, Choose Tax option, and provide a fall back rate.
* Activate the StorePep account by clicking the activation link sent to your email id.
* Enter the Carrier(s) & Store details by loging into StorePep.com
* Now you should be able to see the live rates on the Cart/Checkout pages in the woocommerce store.

== Installation ==

Follow the below instructions to install the plugin on your website: 
1. Download and unzip the latest release zip file.
2. In the Wordpress dashboard, go to Plugins -> Add New -> Upload Plugin and upload the latest zip file.
3. Activate the plugin and start working.

How to create a FREE StorePep account and get Integration ID and Secret Key :
1. Install the plugin and go to WooCommerce –> Settings –> Shipping –> StorePep Shipping
2. Provide your email ID and click "Signup StorePep.com & Get API Keys".This will create a FREE StorePep account and the integration is completed.
3. Activate the StorePep account by clicking the activation link sent to your email id.
4. Enter the Carrier(s) & Store details by loging into StorePep.com

Or 

1. Go to [Storepep](https://ship.storepep.com/) and register yourself for FREE. 
2. On getting an account, navigate to Settings and click on "API Keys".
3. Click on Generate API keys to get the Integration ID and Secret Key.

== Frequently Asked Questions ==

= Can I provide shipping rates from more than one shipping carrier simultaneously using this plugin..? =
Yes, absolutely. Once you enter the Shipping Carrier Account Details in StorePep, you will get the real-time shipping rates from more than one shipping carrier simultaneously. 
= Which shipping carriers does this plugin support..? =
Currently this plugin only support UPS, FedEx, DHL, Stamps.com. However, we are constantly updating the plugin and we can choose to provide support for other shipping carriers in the future.
= Do I have lifetime access to free shipping rates? =
Yes, indeed! You are allowed to make 10,000 shipping API calls/month. That amounts to 333 shipping API calls every day.
